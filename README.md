## ‚ö° Quick Start

### 1. Clone & Install
```bash
git clone https://github.com/TuShArBhArDwA/MiniLink.git
cd MiniLink
npm install
```

### 2. Environment Setup
Create `.env.local`:
```env
DATABASE_URL="postgresql://..."
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=...
CLERK_SECRET_KEY=...
```

### 3. Run It
```bash
npx prisma db push
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) for the magic! üåü
